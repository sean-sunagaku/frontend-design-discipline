# Studio Pulse

## プロダクト概要

Studio Pulse は、小さなプロダクトチーム向けのローンチ計画アプリ。
散らばったリリースタスク、レビューの指摘、進行タイムラインを、一つの静かな作業面にまとめる。

## コア価値

- ローンチ前の混乱を、落ち着いた判断面に変える
- 今日やるべきこと、止まっていること、誰の返答待ちかを一目で把握する
- 騒がしいタスク管理ではなく、密度の高い意思決定の作業面として使う

## 想定ユーザー

- 3〜20 人規模のプロダクト / デザイン / エンジニアリングチーム
- Notion、Slack、Linear、Figma のあいだでローンチタスクが散って困っている人
- 派手なダッシュボードより、静かで集中しやすい運用 UI を好む人

## 主な画面

1. Launch board
   今日の優先事項、停止中タスク、次のレビューをまとめる主画面
2. Review thread
   フィードバックと意思決定ログを時系列で扱う画面
3. Release checklist
   出荷前の確認項目を一覧化する画面

## ブランドの方向性

- 静か
- 精密
- 深夜のスタジオのような空気
- 上質だがラグジュアリーではない
- 宣伝より運用

## UI で避けたいこと

- 汎用 SaaS っぽい KPI カードの並び
- 騒がしいレインボー系ダッシュボード
- 役割のない装飾グラデーション
- もっともらしいだけの企業向け文言
- 説明しすぎるコピー
