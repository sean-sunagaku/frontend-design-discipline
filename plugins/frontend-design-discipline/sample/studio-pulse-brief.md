# Studio Pulse

## 概要

Studio Pulse は、小さなプロダクトチーム向けのローンチ計画アプリ。

散らばったリリースタスク、レビューの指摘、進行タイムラインを、
ひとつの静かな作業面にまとめる。

## 何を作るか

Studio Pulse のランディングページを作る。

## 目的

トライアル登録の促進そのものより、
まず「このプロダクトは他の汎用タスクツールと違う」と一目で感じさせること。

## 想定ユーザー

- 3〜20 人規模のプロダクト / デザイン / エンジニアリングチーム
- Notion、Slack、Linear、Figma のあいだでローンチタスクが散って困っている人
- 派手なダッシュボードより、静かで集中しやすい運用 UI を好む人

## 必須メッセージ

- ローンチ前の判断を、静かに前へ進める
- 散らばったツール群を、一つの落ち着いた判断面にまとめる
- 速さではなく、解像度と落ち着きで勝つプロダクトである

## トーン

- 静か
- 精密
- 深夜のスタジオのような空気
- 上質だがラグジュアリーではない
- 宣伝より運用
