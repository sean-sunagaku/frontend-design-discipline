# レビュー基準

plugin の skill 群を sample LP に対して試すときは、この基準を使う。

## 全体の合格条件

- ファーストビューで Studio Pulse が汎用的ではなく、固有のプロダクトに見えること
- ブランドが静かで精密で運用的に読めること
- 説明に頼らず、階層が明確であること
- コピーが短く、意図的で、無駄がないこと
- 文字が見切れ、白飛び、変な折り返し、壊れかけに見えないこと

## critic ごとの確認項目

### Composition

- 主役が一目で分かること
- セクションが交換可能なカード列に見えないこと
- CTA が stray button ではなく、構図上の結論として置かれていること

### Brand

- ランダムな SaaS ホームページではなく、集中したローンチ作業面に見えること
- ダークトーンと差し色が装飾ではなく、プロダクトの物語を支えていること

### Copy

- 見出しと CTA がすぐ理解できること
- 補助文が実際に役立つか、不要なら消えていること
- 日本語または英語の折り返しが意図的に見えること

### UI 監査

- ダークな面でも文字が無理なく読めること
- モック、レール、境界のある領域に安全な余白があること
- デスクトップとモバイルの両方に適応できそうな構造であること
